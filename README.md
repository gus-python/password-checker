# Password Checker

Script simples em Python para verificar a força de senhas com base em critérios básicos de segurança.

## O que o programa verifica

- tamanho mínimo da senha
- presença de números
- letras maiúsculas
- letras minúsculas
- caracteres especiais

## Tecnologias utilizadas

- Python

## Objetivo do projeto

Projeto desenvolvido para praticar lógica de programação e manipulação de strings em Python.
